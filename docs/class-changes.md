# Class Changes

This page covers the custom class changes made in ValkariaRO. These adjustments were made to improve class balance, encourage more diverse gameplay, and create a healthier experience for PvM, MVP, and WoE while still preserving the original identity of each class.

---
## Swordsman

<h2> Crusader & Paladin </h2>

| Skill | Previous Behaviour | Current Behaviour |
|---|---|---|
| Grand Cross | Official Behaviour | All waves now deal damage to all targets |
| Auto Guard | Official Behaviour | No longer causes stagger when blocking |
| Spear Quicken | Official Behaviour | Can now be used with one-hand spears |
| Defender | Official Behaviour | Duration increased from 180 seconds to 300 seconds |

---

<h2> Knight & Lord Knight </h2>

| Skill | Previous Behaviour | Current Behaviour |
|---|---|---|
| Bowling Bash | Official Behaviour | Area chain reaction now fully follows the official Gutter Line system. Previous demi-gutter bug has been fixed. Targets knocked outside the area will no longer trigger splash damage. |
| Bowling Bash | Official Behaviour | Slightly reduced cast time |
| Onehand Quicken | Official Behaviour | Duration increased from 300 seconds to 600 seconds |
| Parry | Official Behaviour | Duration increased from 60 seconds to 300 seconds |
| Parry | Official Behaviour | Blocking no longer causes stagger during attacks or movement |
| Spiral Pierce | Official Behaviour | Slightly reduced cast time |
| Spiral Pierce | Official Behaviour | Slightly reduced after-cast delay |
